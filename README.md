# AI Chatbot Blueprint

## Project Overview
This project is an AI-powered chatbot that combines Retrieval-Augmented Generation (RAG) with Web Search to provide intelligent answers.

The chatbot can:
- Answer user questions using AI
- Read uploaded PDF documents
- Retrieve relevant information
- Perform web search if information is not found

## Technologies Used
- Python
- Streamlit
- LangChain
- Groq API
- FAISS Vector Database
- DuckDuckGo Search

## Features
- Interactive chatbot interface
- PDF document upload
- Retrieval-Augmented Generation (RAG)
- Web search fallback
- AI-generated responses
- Cloud deployment
